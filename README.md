## Mac版本教程

### 1、下载 .NET SDK Arm64

https://dotnet.microsoft.com/zh-cn/download 

### 2、Release中下载Mac版本

https://github.com/NepPure/YuanshenAnti/releases

> 自己编译一下也行

### 3、控制台启动

定位到`YuanshenAnti.dll`所在目录,先启动脚本

```sh
sudo dotnet YuanshenAnti.dll
```

按提示操作即可
